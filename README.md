## 📖 Overview

**Flask Student Management** is a simple web application developed using the **Flask** framework. It demonstrates the fundamentals of backend web development by combining Python with HTML, CSS, and Jinja2 templates.

This project showcases server-side routing, dynamic template rendering, form processing, redirects, and REST API creation. It is designed as a learning project for beginners who want to understand how Flask applications are structured and built.

---

## ✨ Features

- 🏠 Dynamic Home Page
- 👨‍🎓 Student Information Display
- 📄 About Page
- 📝 Student Form Submission
- ✅ Server-side Form Validation
- 🔄 Page Redirection
- 📡 JSON API Endpoint
- 🎨 Jinja2 Template Rendering
- 📱 Responsive Design

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/archita251105/flask-student-management.git
```

---

## Navigate into the Project

```bash
cd flask-student-management
```

---

## Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

### Activate the Virtual Environment

Windows

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python app.py
```

---

Open your browser and visit:

```
http://127.0.0.1:5000
```
